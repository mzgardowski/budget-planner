# Budget Planner

A modern budget planning application built with React, TypeScript, and Vite. This app helps users manage their finances by tracking income, expenses, and budget allocations.

## Available Commands

### Development
```bash
yarn dev
```
Starts the development server with hot module replacement (HMR).

### Build
```bash
yarn build
```
Compiles TypeScript and builds the application for production.

### Lint
```bash
yarn lint
```
Runs ESLint to check code quality and style.

### Preview
```bash
yarn preview
```
Previews the production build locally.
